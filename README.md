## What is this?

Bron–Kerbosch algorithm to find all maximal cliques in a graph.

## Implementations

* **Ver1:** naive Bron–Kerbosch algorithm
* **Ver2:** Ver1 with pivot
* **Ver3:** Ver2 with degeneracy ordering

## Run
    python maximal_cliques.py
